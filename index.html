What does it mean to ‘cast a double to an int’? Give an example of when you would do this and explain what is happening.

Doubles are a floating point number thus stored in form ""+1.234 x 2 ^ -5". Integers are stored as simple binary numbers. So 5.0 as a float does not have the same bit representation as "5 as an integer". Also the number of bits used to store the answer can differ. Casting allows you to transfer over the "real" value between types with possible losses of data( a risk you maky have to take) For example, if you generate a random number, it is generated as a double(with decimal points). However you may want a random integer. You would need to cast for this to be possible.

What is the difference between ‘while’ and ‘do while’ loops? Give an example of when you would use one versus another. What is the benefit of a ‘do while’ over a ‘while’?

Do while loops execute the code inside once before preforming the boolean check. This makes them useful when you want to always do something at least once. Think of a simple tic-tac-toe game: you may want program to automatically run the game once, and then after the game is over, ask if they want to play again.

What is ‘void’ in a method declaration? How does this differ from declaring a method with a return type?

void simply means that the function will not return a value. Functions(methods) have the ability to return a value. This is useful for a function that takes in two numbers and adds them together. But if you are printing numbers to the command line, you do not need to return any value ergo void.

There is a slight difference between parameter and argument, at least in the usage of the term. Argument is the values that get taken in when the method is called. Parameters are the names of these values within the scope of the method.
